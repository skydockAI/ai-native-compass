# Execution Plan: DI-012 — UI Polish — Modern Light Theme

## Delivery Item

[DI-012](../delivery-items/DI-012.md)

## Objective

Replace the ad-hoc inline styles in `base.html` with a structured design system implemented as a custom CSS file, and update all existing templates to look modern, professional, and user-friendly.

## Design Decisions

- **Primary colour**: `#1565C0` (deep blue, Material Design Blue 800)
- **Supporting palette**:
  - Dark primary: `#0D47A1`
  - Light primary tint: `#E3F2FD`
  - Success teal: `#00897B`
  - Warning amber: `#F57C00`
  - Body background: `#F1F5F9` (slate-100)
  - White surface: `#FFFFFF`
- **Typography**: System font stack (`-apple-system`, `Segoe UI`, `system-ui`)
- **Shadows**: Subtle two-level shadow system (sm / md)
- **Bootstrap**: Kept as the grid and utility foundation; primary colour and component styles overridden with custom CSS

## Steps

1. **Create `src/app/static/css/main.css`**
   - Define CSS custom properties (variables) for the design system
   - Override Bootstrap component styles: navbar, sidebar, cards, alerts, tables, forms, buttons, modals, badges
   - Add `.stat-card` utility class for dashboard summary cards
   - Add `.page-header` utility for consistent page headings

2. **Update `src/app/templates/base.html`**
   - Load `main.css` via `url_for('static', filename='css/main.css')`
   - Remove inline `<style>` block (now covered by `main.css`)
   - Change navbar class from `navbar-dark bg-dark` to `navbar-expand-lg` with `id="main-navbar"` (styled via CSS)
   - Add sidebar section labels ("Navigation", "Administration") for visual grouping

3. **Update `src/app/templates/dashboard/index.html`**
   - Replace solid `text-bg-*` Bootstrap cards with `.stat-card` components
   - Each stat card: icon badge, numeric value, label, optional link

4. **Update `src/app/templates/components/flash_messages.html`**
   - Add appropriate Bootstrap Icons to each alert category (success, danger, warning, info)

5. **Run tests** to confirm no regressions
6. **Start Docker Compose**, verify health check passes
