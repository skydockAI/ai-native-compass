# DI-012: UI Polish — Modern Light Theme

## Summary

Redesign the application UI with a modern, professional light theme using deep blue as the primary colour. This delivery item covers the base layout, navigation, dashboard, and shared UI components.

## Status

In Progress

## Dependencies

- DI-001 (Project Foundation & Infrastructure Setup)

## Requirements Covered

- REQ-055 — Dashboard Overview Page
- REQ-056 — Dashboard Summary Cards
- REQ-057 — Navigation
- REQ-067 — UI Performance & Responsiveness

## Scope

- Create a custom CSS stylesheet (`src/app/static/css/main.css`) that defines the design system: colour palette, typography, spacing, shadows, and component overrides.
- Update `base.html` to apply the new design system across the entire application shell (navbar, sidebar, layout).
- Update `dashboard/index.html` to use modern stat cards with icons.
- Update `components/flash_messages.html` to use left-border accent style.
- No database changes required.

## Out of Scope

- Full dashboard with live data (covered by DI-010)
- Any new routes or business logic
