# REQ-061: Audit Log Access Control

## Priority
High

## Description
Only Admin users can view audit logs. Editor and Viewer users are denied access.

## Acceptance Criteria
- Only users with the Admin role can access the audit log page
- The audit log navigation item is hidden for non-Admin users
- Server-side authorization prevents non-Admin access to audit log data
- Audit logs are view-only; no edit, delete, or restore functionality in v1
