# REQ-031: Repo Template Entity Definition

## Priority
High

## Description
A Repo Template defines the structure (artifacts) used by Repositories assigned to that template.

## Acceptance Criteria
- Repo Template entity includes the following fields:
  - Name (string, required)
  - Description (string, optional)
  - Artifacts (list of artifact definitions)
  - Active status (boolean, required)
  - Created timestamp (datetime, auto-set)
  - Updated timestamp (datetime, auto-updated)
  - Archived status (boolean, required, default false)
- Only Admin users can create, edit, and archive Repo Templates
