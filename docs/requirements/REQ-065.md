# REQ-065: Host-Mounted Persistent Storage

## Priority
High

## Description
Persistent data must be stored on the host machine using Docker volumes or bind mounts mapped to the `.data` directory under the project folder. Data must not depend on container-local storage.

## Acceptance Criteria
- Database data is persisted using a Docker volume or bind mount
- The mount target is the `.data` directory under the project root
- Stopping and restarting containers does not result in data loss
- Data is not stored inside the container's filesystem
