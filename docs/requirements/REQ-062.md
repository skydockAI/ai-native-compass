# REQ-062: Audit Log UI

## Priority
Medium

## Description
The audit log UI must support filtering, search, and view-only access for Admin users.

## Acceptance Criteria
- Audit log page displays log entries in reverse chronological order
- Filtering is supported by:
  - Entity type
  - Action type
  - User who performed the action
  - Date range
- Text search is supported within audit log entries
- Audit log is view-only (no edit, delete, or restore actions)
- Pagination is supported for large result sets
