# REQ-059: Audit Log Events

## Priority
High

## Description
The system must log specific important actions to the audit log.

## Acceptance Criteria
The following events must be logged:
- Login (successful)
- Logout
- Create (for all business entities: User, Team, Product, Repository, Template)
- Update (for all business entities)
- Archive (for all business entities)
- Role changes (user role assignment/modification)
- Template changes affecting live Repositories (artifact add/rename/remove, list option changes)
