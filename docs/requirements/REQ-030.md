# REQ-030: Repository Shared Data Across Products

## Priority
High

## Description
Template-derived data for a Repository is stored once and shared consistently across all linked Products. There is no per-Product override of Repository data.

## Acceptance Criteria
- Repository artifact values are stored at the Repository level, not per Product-Repository link
- All Products linked to the same Repository see identical adoption data
- Editing artifact values on a Repository is reflected for all linked Products immediately
