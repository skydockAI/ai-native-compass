# REQ-005: User Self-Service Password Change

## Priority
Medium

## Description
Authenticated users can change their own password. The user must provide their current password and a new password.

## Acceptance Criteria
- User can access a password change form after login
- Current password must be provided and validated before the change is accepted
- New password must be at least 8 characters long
- New password replaces the old password (stored securely per REQ-003)
- User receives confirmation upon successful password change
- Password change is rejected if the current password is incorrect
- Password change is rejected if the new password does not meet the minimum length requirement
