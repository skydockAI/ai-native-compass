# REQ-052: Repository Duplication

## Priority
Medium

## Description
Editors and Admins can duplicate an existing Repository. The duplicate copies all shared and template-derived values but does not copy Product links by default.

## Acceptance Criteria
- A "Duplicate" action is available on existing Repositories for Admin and Editor users
- Duplicate copies all shared attribute values from the source Repository
- Duplicate copies all template-derived artifact values from the source Repository
- Duplicate does not copy Product links by default
- The user must enter a new unique Repository URL before saving the duplicate
- The duplicate uses the same Template as the source Repository
- The duplicate is created as a new, independent Repository
