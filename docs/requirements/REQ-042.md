# REQ-042: Template Artifact Addition Propagation

## Priority
High

## Description
When a new artifact is added to a Repo Template, it must appear automatically in all active Repositories using that Template, with an empty/null value.

## Acceptance Criteria
- Adding an artifact to a Template immediately makes it visible on all linked active Repositories
- The new artifact appears with an empty or null value on each Repository
- No manual action is required per Repository to see the new artifact
- The change is reflected on Repository detail and edit pages
