# REQ-014: Editor Role Permissions

## Priority
High

## Description
The Editor role can manage operational tracking data (Products, Repositories, Teams) but cannot manage system configuration (users, templates) or view audit logs.

## Acceptance Criteria
- Editor can create, update, and archive Products
- Editor can create, update, and archive Repositories
- Editor can create, update, and archive Teams
- Editor can edit template-derived artifact values for Repositories
- Editor can duplicate Repositories
- Editor can view the dashboard and all data pages
- Editor cannot manage users
- Editor cannot manage Repo Templates
- Editor cannot change a Repository's assigned template
- Editor cannot view audit logs
