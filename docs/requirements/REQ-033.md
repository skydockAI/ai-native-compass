# REQ-033: Template Archive Protection

## Priority
High

## Description
A Repo Template cannot be archived if it is currently used by any active (non-archived) Repository.

## Acceptance Criteria
- The system prevents archiving a Template that is assigned to one or more active Repositories
- When archive is blocked, the UI displays a clear message explaining which active Repositories use the Template
- Once all dependent Repositories are archived, the Template can be archived
