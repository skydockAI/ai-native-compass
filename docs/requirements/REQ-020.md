# REQ-020: Team Name Uniqueness

## Priority
High

## Description
Team names must be unique across the system.

## Acceptance Criteria
- A unique constraint is enforced on the Team name field at the database level
- Creating or renaming a Team to a duplicate name is rejected with a clear error message
