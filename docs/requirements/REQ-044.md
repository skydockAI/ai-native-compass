# REQ-044: Template Artifact Removal Behavior

## Priority
High

## Description
When an artifact is removed from a Repo Template, it disappears from the active Repository UI. Removed values remain available only through audit history, not in the active UI.

## Acceptance Criteria
- Removing an artifact from a Template hides it from all linked Repository detail and edit pages
- Previously stored values for the removed artifact are not displayed in the active UI
- Removed artifact values are preserved in the database for audit history purposes
- Audit log entries referencing the removed artifact remain accessible
