# REQ-036: Default Shared Attribute Protection

## Priority
High

## Description
The four default shared Repository attributes (Name, Team, URL, Description) cannot be removed or renamed by Admin.

## Acceptance Criteria
- The system prevents deletion of default shared attributes
- The system prevents renaming of default shared attributes
- The UI does not offer remove/rename options for default attributes
- Only custom (Admin-added) shared attributes can be modified
