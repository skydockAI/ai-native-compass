# REQ-064: Docker Compose Deployment

## Priority
High

## Description
The application runs as containerized service(s) using Docker Compose. The deployment is suitable for small internal team usage with low to moderate concurrent usage (up to about 20 users).

## Acceptance Criteria
- Application and database run as Docker Compose services
- A single `docker-compose.yml` (or equivalent) defines all services
- The application can be started with a single `docker compose up` command
- The deployment is suitable for internal team use (not public-facing)
- The system supports up to approximately 20 concurrent users
