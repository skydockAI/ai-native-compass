# REQ-006: Admin Password Reset for Users

## Priority
Medium

## Description
Admin users can set a new password for any user account. This enables account recovery when a user forgets their password.

## Acceptance Criteria
- Admin can navigate to a user's profile and set a new password
- The new password must be at least 8 characters long
- The new password is stored securely per REQ-003
- The target user can log in with the new password immediately
- Non-Admin users cannot reset other users' passwords

## Assumptions
- The raw requirements state "Admin can set new password for users" but do not specify whether the user is notified or whether a temporary password mechanism is used. This implementation assumes a direct password set by Admin without notification.
