# REQ-034: Default Shared Repository Attributes

## Priority
High

## Description
All Repositories share a set of default attributes that exist regardless of the assigned Template: Name, Team, URL, and Description. These fields are mandatory for every Repository.

## Acceptance Criteria
- Every Repository has the following default shared attributes: Name, Team, URL, Description
- These attributes are displayed on all Repository forms (create, edit, detail)
- Name, Team, and URL are required; Description is optional
- These default attributes cannot be removed or renamed by Admin
