# REQ-069: Data Integrity Enforcement

## Priority
High

## Description
Unique constraints and dependency rules must be enforced consistently. Archived entities must not appear in active workflows.

## Acceptance Criteria
- Unique constraints are enforced at the database level for:
  - User email
  - Team name
  - Product name
  - Repository URL
  - Template name
- Dependency rules (archive blocking) are enforced consistently (see REQ-049)
- Archived entities are excluded from active selection lists and workflows
- Constraint violations produce clear, user-friendly error messages
