# REQ-043: Template Artifact Rename Propagation

## Priority
Medium

## Description
When an artifact in a Repo Template is renamed, existing Repository values remain attached to that artifact. The rename is cosmetic and does not affect stored data.

## Acceptance Criteria
- Renaming an artifact in a Template updates its display name everywhere
- Existing Repository values for that artifact are preserved without data loss
- The artifact identity is maintained through an internal ID, not by name
