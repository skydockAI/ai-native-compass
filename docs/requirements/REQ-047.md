# REQ-047: Archived Entity UI Visibility

## Priority
High

## Description
Archived entities are not shown in normal active UI views. They are hidden from list pages, dropdowns, and selection controls by default.

## Acceptance Criteria
- Archived entities do not appear in active list views (Products, Repositories, Teams, Templates, Users)
- Archived entities do not appear in dropdown selectors or link dialogs
- Archived entities remain visible in audit logs and history
- The UI may optionally provide a filter/toggle to view archived entities separately
