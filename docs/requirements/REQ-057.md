# REQ-057: Dashboard Summary Cards

## Priority
Medium

## Description
The dashboard homepage must display summary cards showing counts of key active entities.

## Acceptance Criteria
- Dashboard displays the following summary cards:
  - Total active Products
  - Total active Teams
  - Total active Repositories
  - Total active Templates
  - Total active Users
- Counts reflect only active (non-archived) entities
- Basic filtering widgets are provided where practical
