# REQ-041: List Option Lifecycle Management

## Priority
Medium

## Description
For `Other`-type artifacts with value type `list`, Admin defines the available options. List options that are currently in use by active Repositories cannot be deleted but may be deactivated.

## Acceptance Criteria
- Admin can add options to a list-type artifact
- Admin can edit existing list options
- A list option currently used by an active Repository cannot be deleted
- A list option currently used by an active Repository may be deactivated (hidden from new selections but preserved on existing records)
- Unused list options can be deleted
- Deactivated options are still displayed on Repositories that already have them selected but are not available for new selection
