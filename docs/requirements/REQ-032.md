# REQ-032: Template Name Uniqueness

## Priority
High

## Description
Repo Template names must be unique across the system.

## Acceptance Criteria
- A unique constraint is enforced on the Template name field at the database level
- Creating or renaming a Template to a duplicate name is rejected with a clear error message
