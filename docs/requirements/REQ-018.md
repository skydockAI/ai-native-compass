# REQ-018: Admin-Only User Management

## Priority
High

## Description
Only users with the Admin role can manage user accounts. This includes creating, editing, archiving, role assignment, and reactivation of users.

## Acceptance Criteria
- Only Admin users can access the user management pages
- Admin can create new users with email, full name, role, and initial password
- Admin can edit user details (name, role, active status)
- Admin can archive users
- Admin can reactivate archived users
- Non-Admin users (Editor, Viewer) are denied access to user management functions
- Server-side authorization enforces these restrictions on every request
