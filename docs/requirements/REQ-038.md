# REQ-038: Document/Skill/Agent Artifact Values

## Priority
High

## Description
For artifacts of type `Document`, `Skill`, or `Agent`, each Repository stores one of three possible values: `Yes`, `No`, or `N/A`.

## Acceptance Criteria
- When a Repository uses a Template with Document/Skill/Agent artifacts, the value field presents three options: Yes, No, N/A
- The default value for new artifacts is empty/null until explicitly set
- Editors and Admins can update these values on Repository edit pages
- The UI renders these as a selection control (e.g., radio buttons or dropdown)
