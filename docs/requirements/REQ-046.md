# REQ-046: Soft Delete via Archive

## Priority
High

## Description
All business entities use soft delete through archive behavior. Entities are never physically deleted from the database.

## Acceptance Criteria
- The following entities support archive (soft delete):
  - Users
  - Teams
  - Products
  - Repositories
  - Repo Templates
- Archiving sets the archived status to true and active status to false
- Archived entities are retained in the database
- No hard delete functionality is provided in v1
