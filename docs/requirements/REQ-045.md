# REQ-045: List Option Editing and Deactivation

## Priority
Medium

## Description
List options for Other-type artifacts may be edited. Options that are in use by active Repositories cannot be deleted but may be deactivated.

## Acceptance Criteria
- Admin can edit the label/value of existing list options
- A list option in use by an active Repository cannot be deleted
- A list option in use can be deactivated (soft-removed from selection)
- Deactivated options continue to display on Repositories that already selected them
- Deactivated options are not available for new selection
- Attempting to delete an in-use option shows a clear error with an option to deactivate instead
