# REQ-015: Viewer Role Permissions

## Priority
High

## Description
The Viewer role has read-only access to the system. Viewers can see dashboards, Products, Repositories, and template-derived data, but cannot create, edit, delete, or archive any entity.

## Acceptance Criteria
- Viewer can view the dashboard
- Viewer can view Products list and detail pages
- Viewer can view Repositories list and detail pages
- Viewer can view template-derived data on Repositories
- Viewer cannot create, edit, delete, or archive any entity
- Viewer cannot view audit logs
- UI hides or disables action buttons for Viewers
