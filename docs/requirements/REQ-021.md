# REQ-021: Repository-Team Assignment

## Priority
High

## Description
Every Repository must belong to exactly one Team. Team assignment is required during Repository creation.

## Acceptance Criteria
- The Team field is mandatory when creating a Repository
- A Repository is linked to exactly one Team (not zero, not multiple)
- Team can be changed after Repository creation (via edit)
- Only active Teams are available for selection
