# REQ-035: Custom Shared Attributes

## Priority
Medium

## Description
Admin users can add additional shared attributes beyond the defaults. These custom attributes are text-only fields that apply to all Repositories regardless of Template.

## Acceptance Criteria
- Admin can add new shared attributes through the UI
- Custom shared attributes are text-type fields only
- Custom shared attributes appear on all Repository forms across all Templates
- Custom shared attributes can be used for data entry, filtering, and reporting
- Custom shared attributes are optional for Repositories (not required by default)

## Ambiguities
- The raw requirements do not specify whether custom shared attributes can be removed or renamed after creation. This needs clarification.
- The raw requirements do not specify whether custom shared attributes can be marked as required.
