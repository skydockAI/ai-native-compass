# REQ-029: Repository-Template Immutable Assignment

## Priority
High

## Description
Each Repository must be assigned exactly one Repo Template at creation time. The Template cannot be changed after the Repository is created.

## Acceptance Criteria
- Template selection is required during Repository creation
- Once a Repository is created, the Template field is read-only
- The UI does not provide a way to change the Template on an existing Repository
- The API/backend rejects any attempt to change the Template after creation
- Only active Templates are available for selection during creation
