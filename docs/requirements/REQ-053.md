# REQ-053: Product CRUD Operations

## Priority
High

## Description
Products support create, read, update, and archive operations. Product fields are fixed in v1 (Name and Description).

## Acceptance Criteria
- Admin and Editor can create new Products with Name and Description
- Admin and Editor can edit Product Name and Description
- Admin and Editor can archive Products (subject to REQ-026)
- All users can view Products (list and detail pages)
- Product creation and editing occur on separate pages from Repository management
