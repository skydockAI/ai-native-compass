# REQ-037: Template Artifact Structure

## Priority
High

## Description
Each artifact in a Repo Template defines one item that Repositories using that template are expected to track. Artifacts are displayed as a single unified list on Repository detail and edit pages.

## Acceptance Criteria
- Each artifact has the following properties:
  - Type (required): one of `Document`, `Skill`, `Agent`, or `Other`
  - Name (required): text label for the artifact
  - Value specification (required): defines the value type/options (see REQ-038, REQ-039)
  - Description (optional): explanatory text
- Artifacts are managed by Admin as part of the Template definition
- Artifacts are displayed as a flat unified list on Repository pages
