# REQ-013: Admin Role Permissions

## Priority
High

## Description
The Admin role has full access to the system, including all management and configuration capabilities.

## Acceptance Criteria
- Admin can manage users (create, edit, archive, assign roles, reset passwords)
- Admin can manage Products (create, edit, archive)
- Admin can manage Repositories (create, edit, archive, duplicate)
- Admin can manage Repo Templates (create, edit, archive)
- Admin can manage shared Repository attributes
- Admin can manage Teams (create, edit, archive)
- Admin can view audit logs
- Admin can perform all actions available to Editors and Viewers
