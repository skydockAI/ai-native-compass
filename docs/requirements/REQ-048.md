# REQ-048: Permanent Audit History Preservation

## Priority
High

## Description
Audit history is preserved permanently. Archiving an entity does not remove its history. No data retention/purge rules apply in v1.

## Acceptance Criteria
- Audit log entries are never deleted
- Archiving an entity preserves all associated audit entries
- Audit entries for archived entities remain accessible to Admin users
- No automatic purge or retention policy is applied in v1
