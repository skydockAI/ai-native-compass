# TS-003: Authentication & User Session Management

## Overview

Test suite for DI-002. Validates local authentication, session management, active-user enforcement, admin seeding, and seeded-admin protection.

## Coverage

- **Delivery Item**: [DI-002](../delivery-items/DI-002.md)
- **Requirements**: REQ-001, REQ-002, REQ-003, REQ-004, REQ-007, REQ-008, REQ-009, REQ-010, REQ-068
