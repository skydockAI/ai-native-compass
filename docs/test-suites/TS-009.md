## Test Cases

### TC-009-001: Create product with valid name and description
**Requirement:** REQ-023, REQ-053
**Steps:**
1. Log in as Admin or Editor.
2. Navigate to Products → New Product.
3. Enter a valid unique name and optional description.
4. Submit the form.
**Expected:** Product is created and visible on the product list.

---

### TC-009-002: Create product without description (optional field)
**Requirement:** REQ-023
**Steps:**
1. Log in as Admin or Editor.
2. Create a product with a name and no description.
**Expected:** Product is created successfully; description shows as blank.

---

### TC-009-003: Product name uniqueness — reject duplicate name
**Requirement:** REQ-024
**Steps:**
1. Create a product named "Alpha".
2. Attempt to create a second product named "Alpha".
**Expected:** Creation is rejected with a clear error message about duplicate name.

---

### TC-009-004: Product name uniqueness — reject rename to duplicate
**Requirement:** REQ-024
**Steps:**
1. Create products "Alpha" and "Beta".
2. Edit "Beta" and rename it to "Alpha".
**Expected:** Update is rejected with a clear error message.

---

### TC-009-005: Edit product name and description
**Requirement:** REQ-053
**Steps:**
1. Log in as Admin or Editor.
2. Navigate to a product's edit page.
3. Change name and/or description and save.
**Expected:** Changes are persisted and visible on the product detail page.

---

### TC-009-006: Viewer cannot create, edit, or archive products
**Requirement:** REQ-053
**Steps:**
1. Log in as Viewer.
2. Attempt to navigate to `/products/new`, `/products/<id>/edit`, and submit archive form.
**Expected:** All restricted actions return 403 Forbidden.

---

### TC-009-007: Archive a product with no active repository links
**Requirement:** REQ-026, REQ-053
**Steps:**
1. Create a product with no linked repositories.
2. Archive the product.
**Expected:** Product is archived successfully.

---

### TC-009-008: Archive product blocked by active repository link
**Requirement:** REQ-026, REQ-049
**Steps:**
1. Create a product and link it to an active repository.
2. Attempt to archive the product.
**Expected:** Archive is blocked with a message listing the blocking active repository.

---

### TC-009-009: Archive product allowed after unlinking active repositories
**Requirement:** REQ-026
**Steps:**
1. Create a product linked to an active repository.
2. Unlink the repository from the product.
3. Archive the product.
**Expected:** Archive succeeds.

---

### TC-009-010: Archive repository blocked by active product link
**Requirement:** REQ-049
**Steps:**
1. Link an active repository to an active product.
2. Attempt to archive the repository.
**Expected:** Archive is blocked with a message listing the blocking active product(s).

---

### TC-009-011: Archive repository allowed after unlinking active products
**Requirement:** REQ-049
**Steps:**
1. Link a repository to a product.
2. Unlink the product from the repository.
3. Archive the repository.
**Expected:** Archive succeeds.

---

### TC-009-012: Link repository to product from product detail page
**Requirement:** REQ-025, REQ-054
**Steps:**
1. Log in as Admin or Editor.
2. Navigate to a product detail page.
3. Link an available active repository.
**Expected:** Repository appears in the product's linked repositories list.

---

### TC-009-013: Unlink repository from product
**Requirement:** REQ-025, REQ-054
**Steps:**
1. Product has a linked repository.
2. Unlink the repository from the product detail page.
**Expected:** Repository is removed from the linked list; repository data is unchanged.

---

### TC-009-014: Manage product links from repository edit page
**Requirement:** REQ-025, REQ-054
**Steps:**
1. Navigate to a repository edit page.
2. Link and unlink products from the product links section.
**Expected:** Links are updated; product data is unchanged.

---

### TC-009-015: Product created without linking any repository
**Requirement:** REQ-054
**Steps:**
1. Create a product without linking any repository.
**Expected:** Product is created; linked repositories list is empty.

---

### TC-009-016: Repository created without linking any product
**Requirement:** REQ-054
**Steps:**
1. Create a repository without linking any product.
**Expected:** Repository is created; product links section is empty on edit page.

---

### TC-009-017: Product list active/archived toggle
**Requirement:** REQ-053
**Steps:**
1. Create one active and one archived product.
2. View the products list (default).
3. Toggle to show archived.
**Expected:**
- Default view shows only active products.
- Archived view shows only archived products.

---

### TC-009-018: Product detail page shows linked repositories
**Requirement:** REQ-025
**Steps:**
1. Link multiple repositories to a product.
2. View the product detail page.
**Expected:** All linked repositories are listed.

---

### TC-009-019: Reactivate archived product
**Requirement:** REQ-053
**Steps:**
1. Archive a product.
2. Reactivate it.
**Expected:** Product returns to active state.

---

### TC-009-020: Unlinking does not affect repository data
**Requirement:** REQ-025
**Steps:**
1. Link a repository to a product.
2. Unlink the repository.
3. View the repository detail page.
**Expected:** Repository's fields, artifact values, and audit history are unchanged.
